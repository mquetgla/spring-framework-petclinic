## 1. Entidad JPA

### Archivo a crear
`src/main/java/org/springframework/samples/petclinic/model/Vaccine.java`

```java
@Entity
@Table(name = "vaccines")
public class Vaccine extends BaseEntity {

    @Column(name = "name", nullable = false, unique = true)
    @NotBlank(message = "Name is required")
    private String name;

    @Column(name = "manufacturer")
    private String manufacturer;

    @Column(name = "description")
    private String description;

    @Column(name = "immunity_days", nullable = false)
    @NotNull(message = "Immunity days is required")
    @Min(value = 1, message = "Immunity days must be greater than 0")
    private Integer immunityDays;
}
```
